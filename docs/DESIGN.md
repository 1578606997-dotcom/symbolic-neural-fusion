# Symbolic-Neural Fusion (SNF)
## 用结构化符号解决 LLM 长上下文失忆

> 基于 HIL 的符号压缩机制，构建外部结构化记忆层

---

## 核心问题

LLM 的"失忆"现象：
- 10+ 轮对话后开始遗忘早期信息
- 长文档阅读时中间段落提取能力下降
- 复杂推理中前提条件易被遗漏

**根本原因**: Transformer 软注意力随序列长度稀释

---

## 核心洞察

HIL 的压缩特性：
```
自然语言: "分析文档，中文输出，bullet格式，3个要点"
HIL 压缩: "? : $ {z,b} (3)"
压缩率: 35-62%
信息保持: 意图完整，可还原
```

---

## 解决方案

### 架构
```
用户输入
    ↓
HIL 压缩引擎 → 提取关键意图，压缩为符号
    ↓
符号记忆矩阵 → 固定长度结构化表示
    ↓
神经融合层 → 注入 LLM 作为"硬锚点"
    ↓
LLM 推理 → 保持完整上下文感知
```

### 特点
| 特性 | 说明 |
|------|------|
| 高密度 | 固定长度，不随对话稀释 |
| 结构化 | 显式意图表示（动作/对象/约束） |
| 软硬结合 | 符号硬锚点 + 软注意力互补 |
| 可解释 | 符号矩阵可直接阅读 |

---

## 应用场景

1. **长对话保持** - 20+ 轮后仍记得最初需求
2. **长文档理解** - 100页后定位第30页细节
3. **复杂推理链** - 多步证明中记住初始假设

---

## 与现有方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| RAG | 精确召回 | 需向量库、有延迟 |
| Streaming | 简单 | 丢失窗口外信息 |
| **SNF** | 轻量、高密度 | 压缩有信息损失 |
