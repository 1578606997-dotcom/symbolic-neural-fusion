# SNF 框架测试计划

## 测试目标
验证 Symbolic-Neural Fusion (SNF) 框架的核心假设。

## 核心假设
1. HIL 压缩后的信息保持率 > 80%
2. 符号锚点能显著增强长上下文性能
3. 压缩开销 < 注意力节省的收益

## 测试方案

### 测试 1：信息保持率测试
**目标**: 验证 HIL 压缩-还原的语义保持能力

**方法**:
1. 准备 100 条不同长度的自然语言指令
2. 使用 HIL Framework 编码为 HIL 符号
3. 使用 HIL Framework 还原为自然语言
4. 使用 LLM 评估语义相似度
5. 统计平均保持率

**预期**: > 80% 语义保持

---

### 测试 2：长上下文性能测试
**目标**: 验证符号锚点对长对话的影响

**方法**:
1. 准备 20 轮对话场景
2. 对照组：原始对话历史直接输入 LLM
3. 实验组：对话历史压缩为 HIL 符号矩阵后输入
4. 测试：询问第 5 轮的关键信息
5. 对比准确率

**预期**: 实验组准确率显著高于对照组

---

### 测试 3：计算效率测试
**目标**: 验证压缩开销 vs 注意力节省

**方法**:
1. 测量 HIL 压缩时间
2. 测量长文本的注意力计算时间
3. 对比：压缩时间 vs 注意力节省时间
4. 计算收益比

**预期**: 压缩开销 < 注意力节省的 20%
